# PostgreSQL Connection Tester (Python)

A simple utility to verify connectivity to a PostgreSQL database and output connection details.

## Features

-   **Connectivity Check**: Attempts to connect using `psycopg2`.
-   **Version Info**: Prints the PostgreSQL server version.
-   **Env Variable Support**: Reads credentials from `.env` file.

## Installation

1.  **Clone the repository**
2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Create a `.env` file with `POSTGRES_DSN` or specific credentials.
2.  Run the tester:
    ```bash
    python run_tester.py
    ```
